# 5.2. Założenia techniczne

**Język programowania:** Python 3.10+.

**Framework webowy:** FastAPI, Uvicorn, Jinja2 (serwer i szablony GUI).

**Biblioteki NLP:** sentence-transformers (HuggingFace), langdetect. Opcjonalnie: Ollama (lokalny LLM do ekstrakcji tez/claims).

**Web scraping / pobieranie treści:** feedparser (RSS/Atom), httpx, Goose3 (ekstrakcja treści artykułów).

**Wizualizacja:** brak dedykowanych bibliotek (Matplotlib, Plotly, WordCloud nie są używane w projekcie).

**Baza danych:** SQLite (przechowywanie pobranych artykułów, tez, tematów i wyników porównań).

**Inne:** PyYAML (konfiguracja), Click + Rich (CLI), NumPy, SciPy (embeddings, grupowanie, klasteryzacja).
